
# Confess Web Vercel Ready

## Deploy ke Vercel

1. Upload project ke GitHub
2. Import project ke Vercel
3. Tambahkan ENV berikut:

- FONNTE_TOKEN
- TELEGRAM_BOT_TOKEN
- OWNER_TELEGRAM_ID
- DOMAIN_URL

4. Deploy selesai

## Local Run

```bash
npm install
npm start
```

## Features
- Anonymous confess
- Reply confess by ID
- Telegram notification
- Fonnte WhatsApp API
- Vercel ready
